# ML-ticket-dispatcher
Automated Machine Learning ticket dispatch system that predicts ticket team and assigns them to agents/consultants. Built with Python and AWS Services (Lambda, Sagemaker, S3, DynamoDB, EFS). Simulates real-world workflows including notification and agent assignment.
